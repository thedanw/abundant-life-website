# Abundant Life Centre Mareeba — Brand Voice Guidelines

## Brand Essence

**Who we are:** A Pentecostal/Charismatic church (Australian Christian Churches) in Mareeba, Far North Queensland, serving the community since 1983. Led by Ps Darryl & Patricia Roos since 1997.

**Core Scripture:** *"I have come that they may have life, and that they may have it more abundantly."* — John 10:10b (NKJV)

**Tagline:** "Have a heart of worship" / "We exist to serve our community and help others to live in abundance"

**Denomination:** Australian Christian Churches (ACC) — Pentecostal, Spirit-filled, evangelical

---

## Voice Pillars

| Pillar | Definition | Too Much | Just Right | Too Little |
|--------|------------|----------|------------|------------|
| **Welcoming** | Warm, inclusive, inviting — like a family gathering | Overly familiar, boundary-less | "Come and join us" / "You're welcome here" | Cold, institutional, "visitors welcome" signage only |
| **Faith-Filled** | Confident in God's promises, expectant of the supernatural | Presumptuous, "name it claim it" | "We believe God heals" / "Expect a miracle" | Cautious, hedging, "if it be Thy will" as default |
| **Scriptural** | Bible-saturated, chapter-and-verse grounded | Proof-texting, weaponizing verses | "The Bible says..." / "Psalm 47:1 declares..." | Vague spirituality, feelings-based |
| **Expressive** | Demonstrative worship — hands raised, voices lifted, hearts engaged | Performative, emotional manipulation | "Clap your hands, shout to God" / "Lift holy hands" | Stoic, spectator-only, "frozen chosen" |
| **Generous** | Open-handed with time, resources, prayer, community | Enabling, no boundaries | "All proceeds fund missions" / "Bring your needs to prayer" | Transactional, scarcity mindset |
| **Missional** | Outward-focused, equipping saints for works of service | Activism without discipleship | "Equip to minister" / "Reach the world" | Inward-focused, maintenance mode |

## Vocabulary Rules

### Do Use

| Word/Phrase | Why |
|-------------|-----|
| "Family of God" / "Body of Christ" | Core identity metaphors used throughout site |
| "Abundant life" / "Life more abundantly" | Brand anchor from John 10:10 |
| "Minister to" / "Ministry" | Active service language (not passive "serve") |
| "Equip" / "Empower" / "Release" | Ephesians 4:12 language — equipping saints |
| "Gather" / "Gathering" | Hebrews 10:25 — "not forsaking the assembling" |
| "Worship in spirit and truth" | John 4:24 — core worship value |
| "Prophecy" / "Tongues" / "Interpretation" / "Healing" | Specific spiritual gifts named and normalized |
| "Tithe" / "Offering" / "First fruits" | Biblical giving language (Mal 3:10, Prov 3:9) |
| "Pastor" / "Ps" (abbreviation) | Leadership title used consistently |
| "Mareeba" / "Tablelands" / "Far North Queensland" | Geographic identity |
| "ACC" / "Australian Christian Churches" | Denominational affiliation |
| "Livestream" / "Video Church" / "Audio recordings" | Digital ministry terms |
| "Prayer team" / "Intercede" / "Agree together" | Corporate prayer language (Matt 18:19) |

### Don't Use

| Word/Phrase | Why | Alternative |
|-------------|-----|-------------|
| "Audience" / "Spectators" | Passive consumer language | "Congregation" / "Family" / "Gathering" |
| "Performance" / "Show" | Entertainment framing | "Worship" / "Ministry" / "Service" |
| "Self-help" / "Life coaching" | Secular psychology framing | "Discipleship" / "Pastoral care" / "Biblical counseling" |
| "Seeker-sensitive" (as primary label) | Dilutes Spirit-led distinctives | "Welcoming to all, led by the Spirit" |
| "Campus" (for single location) | Multi-site language | "Church" / "Centre" / "133-137 Walsh Street" |
| "Programs" / "Ministries" (as silos) | Institutional feel | "Life together" / "Family rhythms" |
| "Volunteer" (as primary identity) | Reduces calling to task | "Servant" / "Team member" / "Minister" |
| "Customer" / "Client" | Transactional | "Guest" / "Visitor" / "Friend" |
| "Problem" / "Issue" (for prayer needs) | Deficit language | "Need" / "Request" / "Burden" |
| "Fundraising" | Transactional | "Partnering in mission" / "Sowing into kingdom work" |

## Grammar & Style Rules

| Rule | Guideline |
|------|-----------|
| **Spelling** | Australian English (colour, centre, organise, realise) |
| **Scripture References** | Book Chapter:Verse format (e.g., John 10:10, Ps 47:1, 1 Cor 12:8-10); use standard abbreviations |
| **Bible Translation** | NKJV primary (John 10:10b quoted on homepage); KJV language in worship sections (Psalms); NIV/ESV acceptable for clarity |
| **Headings** | Title Case for page titles; Sentence case for section headers |
| **Body Text** | Sentence case |
| **Oxford Comma** | Yes — for clarity in lists of spiritual gifts, beliefs, etc. |
| **Contractions** | Yes — warm, conversational ("we're", "you're", "God's") |
| **Exclamation Marks** | Yes — in worship, celebration, invitation contexts; max 1 per paragraph |
| **ALL CAPS** | Only for: GOD, JESUS, HOLY SPIRIT, BIBLE, ACC (denomination) — reverence/acronyms |
| **Numbers** | Spell out one–nine; numerals for 10+ (except Scripture refs, times, addresses, phone) |
| **Times** | "9:30 am" / "11:30 am" (lowercase am/pm with space) |
| **Dates** | "6th of January 1997" (ordinal) for history; "2nd and 5th Saturday" for recurring |
| **Phone Format** | +61(07)4092 5004 (international format on contact page) |
| **Address Format** | "133-137 Walsh Street, Mareeba, QLD 4880, Australia" |
| **Pastor Title** | "Ps" abbreviation acceptable after first full "Pastor" reference |

## Channel-Specific Notes

### Website (Primary)
- **Homepage:** Invitational, vision-casting, clear next steps (service times, livestream, give, contact)
- **About/History:** Narrative, honoring legacy, showing God's faithfulness across generations
- **What We Believe:** Systematic, Scripture-saturated, catechetical clarity
- **Missions:** Partner-focused, impact storytelling, "equipping the saints" framework
- **Prayer:** Safe, confidential, expectant (Matt 18:19 agreement)
- **Give:** Worship-framed, multiple pathways, transparent
- **Contact:** Practical, accessible, human (office hours, real phone, real email)

### Facebook / Instagram
- **Tone:** Celebratory, behind-the-scenes, community-building
- **Content:** Service highlights, testimony snippets, market day photos, prayer prompts, event reminders
- **Hashtags:** #ALCMareeba #AbundantLife #Mareeba #Tablelands #FNQ #AustralianChristianChurches
- **Frequency:** 3-5x/week (Sun service, midweek encouragement, events, missions, prayer)
- **Voice:** First-person plural ("We had..." "God moved..." "Join us...")

### Livestream / YouTube
- **Pre-service:** Countdown with worship instrumental, welcome slide with John 10:10
- **Host/Hostess:** Warm, directive ("Wherever you're watching, stand with us...")
- **Chat moderation:** Active prayer responses, welcome new viewers, share links (give, prayer, connect)
- **Post-service:** Save to YouTube, timestamp sermon, add description with Scripture refs & links

### Email / Newsletter (if implemented)
- **Subject lines:** Personal, benefit-led ("A word for your week" / "Prayer covered you this Sunday")
- **From name:** "Ps Darryl & the ALC Family" or "Abundant Life Centre Mareeba"
- **Cadence:** Weekly (Monday recap + Wednesday encouragement + Friday preview)
- **Footer:** Always include service times, address, give link, unsubscribe, privacy

### Print / Physical (Bulletins, Signage, Banners)
- **Font:** Clean, readable, warm (serif for Scripture, sans-serif for info)
- **Color:** Brand colors (derive from logo — likely gold/blue/white for "abundant life")
- **Language:** Concise, action-oriented, Scripture-anchored
- **QR codes:** Link to digital connect card, give, prayer, livestream

## Validation Checklist (for new content)

Before publishing, verify:

- [ ] **Scripture-anchored** — At least one direct Bible reference or clear biblical principle
- [ ] **Voice pillars present** — Welcoming + Faith-filled minimum; others as context fits
- [ ] **Vocabulary check** — No "Don't Use" words; brand terms used correctly
- [ ] **Tone matches channel** — Web ≠ Social ≠ Stream ≠ Email
- [ ] **Clear next step** — What should the reader *do*? (Visit, pray, give, connect, watch)
- [ ] **Contact info accurate** — Address, phone, email, service times, give links current
- [ ] **Australian spelling** — Colour, centre, organise, realise, programme
- [ ] **Pastor title** — "Pastor Darryl & Patricia Roos" first ref, then "Ps Darryl & Patricia"
- [ ] **Mission partners named** — IMLA, Living Waters, Gideons, SU QLD, Alphacrucis, ApostleNet
